Where does the Beginner / Advanced mode toggle live?

Issue #84 requires Beginner mode to be the default for first-time users. After that, users need a way to switch. Three approaches:

A

First-run modal + header toggle

On first visit a short modal asks "Are you new to trading bots?" and sets the mode. After that, a small pill toggle in the header lets you switch at any time. Mode saved in localStorage.

Header preview:
Pacekeeper   [DEMO]   [MARKET CLOSED]     Beginner ↔ Advanced
B

Tab-based switcher in the dashboard body

Two tabs at the top of the main content area: "Overview" (Beginner) and "Advanced". No modal — Advanced tab is always accessible but not the default. Mode persists in localStorage.

Content area preview:
Overview Advanced
C

Settings panel only

Mode is controlled inside the existing LLM Settings panel (rename it "Settings"). Not surfaced in the header or as tabs — advanced users who want raw data know where to find it. Beginner mode is default.
